# 《两面来敌》兵种与战斗规则

## 版本变更简述

- **v0.1.0（2026-07-22）：** 建立第一版兵种与战斗规则，明确基础属性、武将重伤、骑乘与落马、远程弹药、装备耐久、阵型克制和战前整备规则。

## 1. 文档目的

本文档用于统一《两面来敌》的战斗术语、兵种属性、状态转换和克制关系，作为后续 Godot 战斗原型、数值配置和战斗结算实现的设计依据。

本版以规则闭环为主。文中数值是第一版基准，不代表最终平衡结果，后续应通过实际战斗测试持续调整。

## 2. 核心设计原则

1. 普通步兵作为数值基准，各项基础属性记为 `1.00`。
2. 兵种强弱由属性、装备、阵型、弹药、移动状态和克制条件共同决定。
3. 克制尽量依赖可观察的战场条件，不使用脱离战场表现的纯随机秒杀。
4. 骑兵必须区分骑乘状态和落马步战状态，两套属性独立配置。
5. 武器、防具、盾牌等装备存在耐久；耐久下降会逐步削弱装备效果。
6. 战前整备按投入比例连续计算，界面可以用三级描述帮助玩家理解，但实际效果不限定为三个固定档位。
7. 武将不会永久死亡；普通兵员可以阵亡，并对主城兵员储备产生长期影响。

## 3. 战斗基础属性

### 3.1 通用属性

| 属性 | 说明 |
| --- | --- |
| 血量 | 单位可承受的伤害总量，归零后普通士兵阵亡或失去战斗能力，武将进入重伤状态 |
| 伤害 | 单次攻击造成的基础伤害 |
| 防御 | 降低所受伤害的基础能力，区别于作为物品存在的“防具” |
| 移动速度 | 单位在战场上的基础移动能力，也是骑兵冲锋伤害的计算参数之一 |
| 攻击范围 | 单位可以发起攻击的有效距离 |
| 攻击速度 | 单位攻击频率；数值越高，单位时间内攻击次数越多 |
| 阻挡数量 | 单位或编队能够同时拦截的敌军数量 |
| 部署费用 | 将部队投入战场所需的局内军资 |
| 再部署时间 | 部队撤离或失去战斗能力后再次部署需要等待的时间 |

### 3.2 兵种专属属性

| 属性 | 适用兵种 | 说明 |
| --- | --- | --- |
| 备箭 | 弓兵、弓骑兵 | 每次远程射击消耗箭矢，耗尽后无法继续射箭 |
| 备弹 | 火枪兵 | 每次射击消耗弹药，耗尽后转入近战状态 |
| 装填进度 | 火枪兵 | 完成装填后才可进行下一次射击，受到干扰可能被打断 |
| 马匹生命 | 各类骑兵 | 归零后触发落马，当前战斗中失去马匹 |
| 冲锋速度 | 各类骑兵 | 达到阈值后进入冲锋状态并获得速度伤害加成 |
| 装备耐久 | 使用装备的兵种 | 决定武器、防具、盾牌等装备当前效果 |
| 阵型完整度 | 长枪兵、盾兵等 | 决定列阵技能和正面防御效果 |

### 3.3 基准值说明

普通步兵的血量、伤害、防御、移动速度、攻击范围和攻击速度均记为 `1.00`。其他兵种的数值是相对普通步兵的倍率。

例如：

```text
弓兵移动速度 = 0.95
```

表示弓兵基础移动速度为普通步兵的 95%，不表示固定减少某个绝对速度值。

远程兵种的实际射程需要在地图单位确定后单独配置，不应直接将近战单位的 `1.00` 当作所有射程计算的绝对距离。

## 4. 武将战斗规则

### 4.1 武将重伤

武将是永久角色，不会在战斗中永久死亡。武将血量归零后进入“重伤”状态：

- 立即退出当前战斗。
- 当前单局无法再次部署。
- 当前单局无法继续使用专属技能、特技和专属战场效果。
- 其统领部队失去武将提供的持续统率效果；已经结算到部队基础属性中的熟练度加成是否保留，由具体技能配置决定。
- 战斗结束后进入治疗状态，治疗完成前不能再次出战。

第一版不实现武将永久死亡、被俘和失联。

### 4.2 武将装备栏

每名武将拥有以下携带栏位：

| 栏位 | 数量 | 可携带内容 |
| --- | ---: | --- |
| 武器栏 | 1 | 刀、长枪、弓、火枪等 |
| 防具栏 | 1 | 轻甲、重甲、盾牌等防护装备 |
| 马匹栏 | 1 | 轻型马、战马、重型战马等 |
| 道具栏 | 2 | 一次性回血道具、临时补给或特殊战场道具 |

马匹统一作为独立物品类型，暂不细分鞍具、马镫等马具组件。

### 4.3 武将重伤与装备损失

| 战斗结果 | 武器 | 防具 | 马匹 | 消耗品 |
| --- | --- | --- | --- | --- |
| 正常获胜 | 少量耐久损耗 | 少量耐久损耗 | 保留 | 按实际使用扣除 |
| 主动撤退 | 少量耐久损耗 | 少量耐久损耗 | 保留 | 按实际使用扣除 |
| 武将重伤 | 中量耐久损耗 | 中量耐久损耗 | 概率丢失 | 未使用部分保留 |
| 全军溃败 | 大量耐久损耗并有概率遗失 | 大量耐久损耗并有概率遗失 | 高概率丢失 | 高概率遗失 |

“武将被击杀”统一解释为武将重伤或部队全军溃败，不代表武将永久死亡。

## 5. 武将兵种适配与熟练度

### 5.1 适配度

武将出战前选择本场统领的兵种。每名武将对不同兵种拥有先天适配度，推荐使用 `S、A、B、C、D` 五档。

适配度影响：

- 熟练度获取速度。
- 熟练度等级上限。
- 部队属性加成效率。
- 兵种专属能力或阵型的解锁条件。

### 5.2 熟练度

熟练度代表武将实际统领某兵种的能力，可以通过以下方式提高：

- 在主城消耗金额和训练时间进行锻炼。
- 实际统领对应兵种完成战斗。
- 使用兵书、教官等特殊资源。

第一版使用五级熟练度：

| 等级 | 名称 | 基础加成 |
| --- | --- | ---: |
| 1 | 生疏 | 0% |
| 2 | 熟悉 | +3% |
| 3 | 熟练 | +6% |
| 4 | 精通 | +10% |
| 5 | 宗师 | +15% |

熟练度采用加算增强，但不同属性使用不同权重：

```text
血量、伤害、防御：基础加成 × 100%
移动速度、攻击速度：基础加成 × 50%
攻击范围：基础加成 × 25%
```

该限制用于避免高熟练度骑兵速度、冲锋伤害或远程射程过度增长。

## 6. 骑乘、冲锋与落马规则

### 6.1 双状态配置

轻骑兵、重骑兵和弓骑兵均必须拥有两套独立属性：

1. **骑乘状态：** 使用马匹属性，享受移动速度与冲锋伤害加成。
2. **落马状态：** 失去马匹，切换为对应的步战属性。

落马属性不是通过骑乘属性临时扣减计算，而是读取对应兵种的独立步战配置。

### 6.2 骑乘速度伤害加成

轻骑兵、重骑兵和弓骑兵在骑乘状态下均可享受速度伤害加成。达到最低冲锋速度后进入冲锋状态：

```text
冲锋伤害 = 基础攻击伤害
         × 当前速度系数
         × 马匹冲击系数
         × 武器系数
         × 兵种克制系数
         × 命中位置系数
```

- 未达到最低速度时，不获得冲锋伤害。
- 转向、被阻挡、受到减速或陷入近战会降低速度并中断冲锋。
- 骑兵需要重新拉开距离并加速，才能再次获得完整冲锋加成。
- 弓骑兵同样享受速度伤害加成，但具体作用于近战冲击、骑射或两者，应由武器与攻击动作配置决定；第一版推荐主要作用于骑乘近战冲击，骑射只获得较低比例加成。

### 6.3 落马触发

以下情况可以触发落马：

- 马匹生命归零。
- 被完整长枪阵正面拦截。
- 被长刀步兵命中马匹并触发斩马伤害。
- 高速撞击重型障碍物。
- 特定技能直接攻击马匹。

### 6.4 落马后果

- 当前战斗失去马匹。
- 立即切换为对应兵种的落马步战属性。
- 根据落马前速度和单位总负重受到额外伤害。
- 高概率进入眩晕状态。
- 轻甲单位眩晕时间较短，重甲单位眩晕时间较长。

```text
落马眩晕时间 = 基础眩晕时间
             + 防具重量修正
             + 当前速度修正
             + 地形修正
```

## 7. 兵种规则

### 7.1 普通步兵

普通步兵是全部兵种的数值基准：

| 血量 | 伤害 | 防御 | 移动速度 | 攻击范围 | 攻击速度 |
| ---: | ---: | ---: | ---: | ---: | ---: |
| 1.00 | 1.00 | 1.00 | 1.00 | 1.00 | 1.00 |

特点：

- 属性均衡。
- 招募和整备成本较低。
- 没有明显克制，也没有严重弱点。
- 适合填补防线和承担常规战斗任务。

### 7.2 轻骑兵

#### 骑乘状态

- 使用马匹和长枪。
- 具有高机动性和速度伤害加成。
- 对未列阵的普通步兵具有冲阵优势。
- 远距离受到火枪兵压制，成功贴近后反过来克制火枪兵。
- 被长枪阵和长刀斩马能力克制。

轻骑兵骑乘属性独立配置，在原型测试阶段根据马匹类型、冲锋距离和转向成本进行平衡，不直接从落马属性推算。

#### 落马步战状态

| 血量 | 伤害 | 防御 | 移动速度 | 攻击范围 |
| ---: | ---: | ---: | ---: | ---: |
| 1.00 | 0.90 | 1.00 | 0.90 | 1.20 |

落马后失去冲锋和速度伤害加成，但仍可凭借长枪进行普通步战。

### 7.3 重骑兵

#### 骑乘状态

- 使用重型战马、长枪和重甲。
- 具有高冲击、高伤害和高防护。
- 骑乘属性独立配置，应保持快于普通步兵，但转向、再次加速和部署成本明显高于轻骑兵。
- 正面冲击未列阵部队时效果显著。
- 被完整长枪阵拦截时会受到严重反冲锋伤害。

#### 落马重甲步战状态

原定“速度低 `0.40`”等属性均指落马后的重甲步战状态：

| 血量 | 伤害 | 防御 | 移动速度 | 攻击范围 |
| ---: | ---: | ---: | ---: | ---: |
| 1.00 | 1.30 | 1.50 | 0.60 | 1.20 |

- 落马后移动缓慢，但仍具有较强近战伤害和防护。
- 因重甲负重较高，落马时受到的眩晕通常比轻骑兵更长。
- 重甲耐久下降时，防御和减伤效果同步衰减。

### 7.4 弓骑兵

#### 骑乘状态

- 相当于骑乘作战的弓兵，拥有独立骑乘属性。
- 骑乘状态下同样享受移动速度与速度伤害加成。
- 可以依靠机动性保持距离并进行远程攻击。
- 骑乘时无法贴近尸体或地面回收备箭。
- 备箭耗尽后可以撤离补给、下马近战或继续以马匹进行机动作战。

#### 落马步战状态

- 落马后使用独立步战属性。
- “攻击范围比步兵略高”仅描述落马后的状态，不描述骑乘射程。
- 落马后攻击范围暂定为普通步兵的 `1.05`。
- 其他属性原则上继承弓兵步战配置，再通过原型测试决定是否增加落马惩罚。

### 7.5 弓兵

#### 持弓远程状态

- 拥有弓和备箭。
- 每次射击消耗一支箭。
- 命中目标后，箭矢可能损毁、留在目标身上或落在战场。
- 目标死亡后，弓兵靠近时可以回收未损毁箭矢。
- 无法接近回收点或备箭耗尽时，不能继续远程攻击。

#### 无箭近战状态

| 血量 | 近战伤害 | 防御 | 移动速度 | 近战范围 |
| ---: | ---: | ---: | ---: | ---: |
| 1.00 | 0.95 | 0.85 | 0.95 | 0.90 |

弓兵的近战能力弱于普通步兵，避免其同时拥有远程与近战双重优势。

### 7.6 火枪兵

#### 远程状态

- 拥有火枪、备弹和装填进度。
- 远程攻击距离暂定为弓兵的 `1.50` 倍。
- 对重甲兵和正面持盾的盾兵以外的大多数兵种具有远程优势。
- 射击后必须完成装填才能再次开火。

#### 装填打断

火枪兵装填过程中发生以下情况时，装填会被打断：

- 受到近战攻击。
- 进入眩晕、击退或倒地状态。
- 主动移动或更换阵地。
- 执行其他需要双手的战斗动作。

被打断后，第一版推荐保留部分装填进度，而不是每次完全归零：

```text
保留装填进度 = 当前装填进度 × 打断保留系数
```

普通受击建议保留 30%～50%，眩晕或倒地可以直接归零。具体系数进入数值配置。

#### 缺弹近战状态

| 血量 | 近战伤害 | 防御 | 移动速度 | 近战范围 |
| ---: | ---: | ---: | ---: | ---: |
| 1.00 | 0.90 | 0.90 | 1.00 | 1.00 |

### 7.7 盾兵

盾兵携带具有独立耐久的盾牌：

| 血量 | 伤害 | 防御 | 移动速度 | 攻击范围 |
| ---: | ---: | ---: | ---: | ---: |
| 1.00 | 0.90 | 1.00 | 0.95 | 0.95 |

- 正面受到远程攻击时获得高额减伤。
- 盾牌预计可以有效抵挡火枪攻击 2～5 次、弓箭攻击 5～9 次，最终次数由盾牌品质、攻击强度和命中位置决定。
- 侧面和背面攻击不享受完整盾牌减伤。
- 盾牌耐久归零后触发“盾破”，单位继续战斗，但失去盾牌减伤。

### 7.8 重甲兵

重甲兵携带重甲：

- 对普通近战、弓箭及部分火枪攻击提供高额减伤。
- 移动速度较慢，招募和整备成本较高。
- 能降低骑兵武器造成的直接伤害，但无法完全抵消高速冲锋产生的冲击，可能被击倒或长时间眩晕。

重甲减伤随耐久逐步降低，不采用“耐久未归零时效果完全不变”的规则：

```text
当前重甲减伤 = 重甲基础减伤
             × 耐久效果曲线
```

第一版建议使用分段连续曲线：

| 剩余耐久 | 减伤效果系数 |
| ---: | ---: |
| 76%～100% | 100% |
| 51%～75% | 90%～100% |
| 26%～50% | 70%～90% |
| 1%～25% | 30%～70% |
| 0% | 0% |

区间内按剩余耐久比例插值，避免效果突然跳变。

### 7.9 长枪兵

| 血量 | 伤害 | 防御 | 移动速度 | 攻击范围 | 攻击速度 |
| ---: | ---: | ---: | ---: | ---: | ---: |
| 1.00 | 1.00 | 1.00 | 0.95 | 1.20 | 0.85 |

长枪兵主要通过完整阵型阻止正面骑兵冲锋：

- 面向来袭骑兵并保持阵型时进入“拒马”状态。
- 阵型被侧击、背击、击退或近战扰乱后，反骑效果下降。
- 不使用固定概率直接秒杀高速骑兵。
- 骑兵、马匹和长枪兵所受伤害，根据双方速度、长枪阵型、命中位置和装备状态计算。

推荐计算框架：

```text
反冲锋伤害 = 长枪基础伤害
           × 骑兵相对速度系数
           × 长枪阵型系数
           × 命中位置系数
           × 长枪耐久系数
           × 兵种克制系数
```

主要参数：

- **相对速度：** 骑兵速度越高，冲击更强，但撞上完整枪阵时承受的反冲锋伤害也越高。
- **阵型完整度：** 完整正面枪阵获得最高反骑系数；零散长枪兵效果明显降低。
- **命中位置：** 命中马匹可能造成落马；命中骑手主要造成单位伤害；命中防具会受到防具减伤。
- **接敌角度：** 正面效果最高，侧面次之，背面通常无法触发完整拒马。

### 7.10 长刀步兵

长刀步兵是主动斩马单位：

- 对骑兵和马匹造成额外伤害。
- 适合攻击已经减速、被阻挡或陷入近战的骑兵。
- 不适合正面承受完整速度的骑兵冲锋。
- 命中马匹时可以触发高额斩马伤害；是否击杀马匹由伤害与马匹剩余生命决定，而非无条件随机秒杀。

长枪兵负责正面阻止冲锋，长刀步兵负责在骑兵失速后斩杀马匹，两者定位不得重叠。

## 8. 远程弹药与回收

### 8.1 箭矢

箭矢具有以下结果：

- 射空后落在战场，可尝试回收。
- 命中未死亡目标时，有概率损毁或留在目标身上。
- 目标死亡后，附近弓兵可以回收未损毁箭矢。
- 骑乘状态的弓骑兵不能执行贴近回收动作。

第一版实现时，可以使用“可回收箭矢计数”代替逐支生成实体，避免大量场景对象影响性能。

### 8.2 弹药

- 火枪弹药使用后不可从尸体直接回收。
- 火枪兵只能通过战前携带、战场补给点或补给技能补充备弹。
- 备弹耗尽后切换为近战状态。

## 9. 装备耐久与战场效果

### 9.1 通用耐久状态

| 剩余耐久 | 状态 | 通用效果建议 |
| ---: | --- | --- |
| 51%～100% | 正常 | 保持主要属性，具体装备可轻微衰减 |
| 21%～50% | 磨损 | 属性和特殊效果逐步降低 |
| 1%～20% | 严重损坏 | 属性明显降低，可能无法触发部分特效 |
| 0% | 损毁 | 不再提供对应装备效果 |

### 9.2 不同装备的损坏结果

- 武器损毁：无法继续使用武器专属攻击，转入徒手或备用武器状态。
- 弓损毁：不能继续进行弓箭攻击，即使仍有备箭。
- 火枪损毁：不能继续射击，即使仍有备弹。
- 盾牌损毁：失去正面远程减伤。
- 重甲损毁：重甲减伤降为零，但重量惩罚是否保留由损坏表现决定；第一版推荐仍保留部分重量惩罚，直到撤离或更换装备。

## 10. 战前整备系统

### 10.1 整备内容

玩家在出征或防守前，可以消耗金额提高本场部队的：

- 武器质量和初始耐久。
- 防具质量和初始耐久。
- 箭矢携带数量。
- 火枪弹药携带数量。
- 马匹状态与冲锋能力。
- 医疗物资。
- 粮草及其他辎重。

### 10.2 比例投入

整备效果按实际投入比例连续计算，不限定为三个固定等级：

```text
整备比例 = 本次投入金额 / 当前项目满额整备金额
```

整备比例限制在该项目允许的最低值和最高值之间。具体效果使用带收益递减的曲线：

```text
整备效果 = 最低效果
         +（最高效果 - 最低效果）
         × 收益曲线（整备比例）
```

界面可以显示以下描述，但描述只用于帮助玩家理解：

- 基础整备
- 充足整备
- 精良整备

同一描述区间内仍按实际投入比例产生不同效果。

### 10.3 整备边界

- 每项整备存在投入上限和效果上限。
- 整备主要作用于本场战斗，不永久提高兵种基础属性。
- 永久成长来自武将熟练度、主城设施、兵种训练、军制和长期装备。
- 高投入应当收益递减，避免金额投入直接形成无限战斗力。

## 11. 兵种克制关系

```text
骑兵冲阵
  → 克制未列阵步兵和暴露的远程单位

长枪列阵
  → 正面克制高速冲锋骑兵

长刀步兵
  → 克制减速、被阻挡或陷入近战的骑兵

弓兵
  → 克制无盾轻甲单位

火枪兵
  → 远程克制多数中轻甲单位和接近前的骑兵

盾兵
  → 正面克制弓兵和火枪兵

重甲兵
  → 克制普通近战和弓箭，但会受到高速冲击造成的击倒与眩晕

普通步兵
  → 依靠低成本和均衡属性填补阵线
```

克制关系必须保留条件和反制空间：

- 火枪兵远距离压制骑兵，但装填会被打断，被贴近后处于劣势。
- 长枪兵正面列阵克制骑兵，但被侧击或阵型被打乱后效果下降。
- 盾兵正面克制远程，但侧面、背面及盾破后防御效果降低。
- 重骑兵冲击重甲兵可以造成眩晕，但撞击完整枪阵会承受更高反冲锋伤害。

## 12. 战场状态定义

| 状态 | 说明 |
| --- | --- |
| 冲锋 | 骑兵达到速度阈值，获得速度伤害加成 |
| 列阵 | 单位保持有效阵型，获得盾阵或拒马等能力 |
| 落马 | 马匹失效，骑兵切换为步战状态 |
| 眩晕 | 暂时不能移动、攻击或执行技能 |
| 重伤 | 武将退出本场战斗，不能再次部署和使用专属能力 |
| 缺箭 | 弓兵无法继续远程射击 |
| 缺弹 | 火枪兵无法继续远程射击 |
| 装填中断 | 火枪兵当前装填被干扰，需要保留部分进度或重新装填 |
| 盾破 | 盾牌耐久归零，失去盾牌防御效果 |
| 甲损 | 防具耐久下降，减伤效果按耐久曲线降低 |
| 溃败 | 部队失去组织并退出战斗，提高装备和物资遗失风险 |
| 整备 | 出战前通过金额投入获得的本场临时强化 |

## 13. 第一版实现建议

### 13.1 优先进入首个战斗原型

- 普通步兵。
- 轻骑兵及骑乘、冲锋、落马状态。
- 弓兵及备箭状态。
- 火枪兵及备弹、装填、装填打断状态。
- 盾兵及盾牌耐久。
- 长枪兵及阵型化反冲锋计算。
- 武将重伤且单局不可再次部署。
- 防具耐久衰减与战前比例整备。

### 13.2 原型稳定后加入

- 重骑兵及落马重甲步战状态。
- 弓骑兵及骑乘、落马两套远程规则。
- 重甲兵的连续耐久减伤曲线。
- 长刀步兵的斩马伤害。
- 战后装备遗失、修理和回收结算。

此处是实现顺序，不表示后四种兵种被永久排除在第一版游戏内容之外。

## 14. 数值配置要求

以下参数必须配置化，不得硬编码在单位脚本中：

- 兵种两套或多套状态属性。
- 骑兵最低冲锋速度、速度伤害曲线和转向惩罚。
- 马匹生命、重量和冲击系数。
- 落马伤害与眩晕时间。
- 弓兵备箭数量、箭矢损毁率和回收率。
- 火枪兵备弹数量、装填时间和打断保留系数。
- 盾牌与重甲的耐久、减伤曲线。
- 长枪阵型系数、命中位置系数和反冲锋伤害。
- 战前各整备项目的金额上限、效果上限及收益曲线。
- 武将熟练度等级与各属性加成权重。

所有战斗记录必须保留本局使用的配置版本，便于服务端结算校验和后续数值回溯。

## 15. 后续待细化事项

1. 各骑乘兵种的正式骑乘属性数值。
2. 地图单位确定后的弓兵、弓骑兵和火枪兵实际射程。
3. 普通单位的编队规模、阻挡数量和阵型人数要求。
4. 马匹受到攻击时，骑手与马匹之间的伤害分配。
5. 武将重伤时马匹的具体丢失概率。
6. 不同品质装备的最大耐久、维修费用和回收比例。
7. 箭矢回收是否需要弓兵主动移动，还是由战斗区域结束时统一结算。
8. 火枪装填被普通远程伤害命中时是否中断。
9. 战前整备收益曲线使用平方根、分段线性还是其他公式。
10. 弓骑兵速度伤害加成对骑射的具体生效比例。
