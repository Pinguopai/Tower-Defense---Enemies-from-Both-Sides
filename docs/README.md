# 《两面来敌》项目文档

## 文档说明

本目录用于维护《两面来敌》（暂定名）的游戏需求、玩法设计与技术方案。

当前版本为第一版草案，目标是明确游戏核心体验、首版范围及客户端—服务端总体架构，为后续原型制作和任务拆分提供依据。

## 当前技术框架

```text
客户端：Godot（上帝视角、类 3D 视觉表现）
    ↓ HTTP / WebSocket
后端：Spring Boot（Java）
    ↓ MyBatis-Plus
数据库：MySQL + Redis
    ↓
运维：Docker + Nginx
```
